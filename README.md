## ğŸ¦‹ Steps to create a new project

- [ ] Create a new repository on GitHub with the name of your project.
- [ ] Clone the repository to your local machine.
- [ ] Copy the contents of the template repository to your project repository.
- [ ] Open the project in your favorite IDE.
- [ ] Open the `pubspec.yaml` file and change the name of the project.
- [ ] Open the `lib/utils/constants.dart` file and change the app name & Primary Color
- [ ] Update App Name at info.plist, AndroidManifest.xml
- [ ] Updated the logo in `assets/images/logo.png`
- [ ] Run command `flutter pub get` to install all the dependencies.
- [ ] Run command `flutter pub run flutter_launcher_icons` to generate the icons.
- [ ] Run command `flutter pub run flutter_native_splash:create` to generate the splash screen
- [ ] Add Firebase to you Project.
- [ ] Run the project and check if everything is working fine.
